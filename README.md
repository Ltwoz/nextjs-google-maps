## Next.js Google Maps

This repository is an example of Google Map widget using `@react-google-maps/api` in Next.js

<hr>

### What this app can do

- Show map widget
  
- Add marker
  
- Remove marker
  
- Get coordinates by url
  
- Get coordinates by input lat, lng

### Getting Started

Clone this repository first then

```bash
pnpm install
# then
pnpm run dev
```
